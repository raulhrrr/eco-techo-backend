# Backend en Nest

## Ejecución local

1. Copiar el archivo `.env.template`, renombrarlo a `.env` y reemplazar los valores.

2. Ejecutar el siguiente comando:

```
npm run start:dev
```

## Ejecución en Docker

1. Copiar el archivo `.env.template`, renombrarlo a `.env` y reemplazar los valores.

2. Ejecutar el siguiente comando:

```
docker compose --env-file .env up -d --build
```
